# Offline AI-Powered Volume Calculator

## Project Overview

The Offline AI-Powered Volume Calculator is a desktop application designed to provide accurate volume calculations for various geometric shapes. Utilizing advanced AI algorithms, this calculator offers a user-friendly interface, making complex calculations simple and accessible to everyone. Whether you're a student, educator, or professional, this tool can help with your volume calculation needs.

## Features
- **AI-Powered Calculations:** Leverages machine learning algorithms for accurate results.
- **User-Friendly Interface:** Intuitive design that simplifies the calculation process.
- **Support for Multiple Shapes:** Calculate volume for cubes, spheres, cylinders, and more.
- **Offline Functionality:** Operates without the need for an internet connection.
- **Cross-Platform Compatibility:** Available on Windows, macOS, and Linux.

## Usage Examples
- **Calculating the Volume of a Sphere**: Input the radius, and the application will return the volume based on the formula \( V = \frac{4}{3} \pi r^3 \).
- **Calculating the Volume of a Cylinder**: Input the radius and height to get the volume using the formula \( V = \pi r^2 h \).

## API Reference
- **Function: calculateSphereVolume(radius)**
  - **Parameters**: `radius` (number) - The radius of the sphere.
  - **Returns**: Volume of the sphere as a number.

- **Function: calculateCylinderVolume(radius, height)**
  - **Parameters**: `radius` (number), `height` (number) - The radius and height of the cylinder.
  - **Returns**: Volume of the cylinder as a number.
